Sandwich Roulette

Overview

Sandwich Roulette is a turn based
luck and strategy game
inspired by Russian Roulette,
players take turns eating sandwiches
some of which are poisoned.

















End Users

This game is targetted towards teens, gamblers and mainly gamers aged 13+
who want to play a game involving both luck and strategy

















Requirements

Number Requirement Name Specifications
1. Items
  • Poison detector (Item)
  • Poison Medicine (Item)
  • Max inventory of four items (System)
2. Luck/Gambling
  • Random poisoned/normal sandwich generator (System)
  • Random item generator (System)
  • Strategies involving luck
3. Turn based system
  • On a players turn they can eat or pass the sandwich, if a poisoned one is eaten the player loses health if a normal one is eaten whoever ate it chooses what to do with the next sandwich (system)
  • Three rounds per game where items and health are reset (System)
  • A turn based system allowing for strategies
4. Winning/Losing system
  • Damage system
  • Healing system
  • Health bar system
















Implications


Aesthetics

The game will have a look similar to the "Buckshot Roulette" game available on steam which has a pixel (dot) style
it will also have a menu similar to the one in the popular game "GTA 5"

Intellectual Property

We will need to be careful with our planned aesthetics of having our game be similar to "Buckshot Roulette" in theme/color,
and having a similar menu to "GTA 5". In order to make it not too similar to already pre existing games,
changes to the theme, color and size of the font and the font will be added to avoid trouble.

Ethical

A game about gambling with death involved poisoned food isn't ethical for a young audience to play,
we will be adding a PG 13+ styled button within the main menu to warn players,
If the button is clicked a warning of potential violence will be shown.

1. Task One

Random Sandwich System

Random Sandwich System

Under the core i defined the getsandwiches function which is the random sandwich system, it makes an empty list of sandwiches.
It is then filled with zeros and ones to differentiate between poisoned and normal sandwiches,
the amount of sandwiches and poisoned sandwiches will always be the same as the round number
so for example if its the second round there will 100% be two normal and two poisoned sandwiches in the list.

Having an entirely preset list of sandwiches would eventually get boring and repetitive so i decided to include random sandwich generation.
for example if it is round two the randint code will generate two random numbers between 0 or 1 ensuring the list can be different.
to finish off the function the list is shuffled with random.shuffle and finally the resulting variable is returned.
